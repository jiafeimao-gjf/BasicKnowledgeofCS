# 129、求根节点到叶子节点的数字之和
给你一个二叉树的根节点 root ，树中每个节点都存放有一个 0 到 9 之间的数字。
每条从根节点到叶节点的路径都代表一个数字：

例如，从根节点到叶节点的路径 `1 -> 2 -> 3` 表示数字 123 。
计算从根节点到叶节点生成的 所有数字之和 。

叶节点 是指没有子节点的节点。

 

## 示例 1：

```
输入：root = [1,2,3]
输出：25
解释：
从根到叶子节点路径 1->2 代表数字 12
从根到叶子节点路径 1->3 代表数字 13
因此，数字总和 = 12 + 13 = 25
```
## 示例 2：

```
输入：root = [4,9,0,5,1]
输出：1026
解释：
从根到叶子节点路径 4->9->5 代表数字 495
从根到叶子节点路径 4->9->1 代表数字 491
从根到叶子节点路径 4->0 代表数字 40
因此，数字总和 = 495 + 491 + 40 = 1026
```

### 提示：

- 树中节点的数目在范围 `[1, 1000]` 内
- `0 <= Node.val <= 9`
- 树的深度不超过 10


> 链接：https://leetcode.cn/problems/sum-root-to-leaf-numbers

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public int sumNumbers(TreeNode root) {
        // 先序遍历，判断是否为叶子节点，若为叶子节点，生成数字，存储到数组之中
        // 累加数组
        if (root == null) {
            return 0;
        }

        return preOrder(root, 0);
    }

    private int preOrder(TreeNode root,int num) {
        if (root == null) {
            return 0;
        }
        if (root.left == null && root.right == null) {
            return num*10 + root.val;
        }
        // int left = 0;
        // if (root.left != null) {
        //     left = preOrder(root.left, num*10 + root.val);
        // }
        // int right = 0;
        // if (root.right != null) {
        //     right = preOrder(root.right, num*10 + root.val);
        // }
        
        // return left + right;
        return preOrder(root.left, num*10 + root.val) + preOrder(root.right, num*10 + root.val);
    }
}
```
