## 题目描述
输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。
### 思路分析
- 辅助变量，只有1位是1，
- 循环，直至辅助变量为0
    - 与运算，统计1的个数
    - 辅助变量中的1左移
```c++
class Solution {
public:
     int  NumberOf1(int n) {
         int count = 0;
         unsigned int flag = 1;
         while(flag){
             //++count;
             //n = (n - 1) & n;
             if (flag & n) ++ count;
             flag = flag << 1;
         }
         return count;
     }
};

```
